# Font Sizes

Mit 12pt sind die Fondgrößen folgende:

| command       | size  | 
|---------------|-------|
| \tiny         | 6pt   |  
| \scriptsize   | 8pt   |  
| \footnotesize | 10pt  | 
| \small        | 11pt  | 
| \normalsize   | 12pt  | 
| \large        | 14pt  | 
| \Large        | 17pt  | 
| \LARGE        | 20pt  | 
| \huge         | 25pt  | 
| \Huge         | 25pt  |
